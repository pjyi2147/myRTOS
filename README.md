## myRTOS

### Real-Time OS for ARMv8 architechture with QEMU simulator

---

### How to run

1. Install requirements: `QEMU`, `gcc-arm-none-eabi`, `make`
2. Run `make` to build kernel
3. Run kernel with QEMU using `make run`

### Features

1. Booting
2. UART
3. Interrupt
4. Timer
5. Task
6. Scheduler
7. Context Switching
8. Event
9. Messaging
10. Synchronization
